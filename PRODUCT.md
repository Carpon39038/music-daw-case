# 🎯 music-daw-case 产品文档

> **定位**：面向零基础用户的浏览器端 DAW，帮助新手 5 分钟内做出自己的小 demo。

---

## 核心原则

1. **新手优先** — 每个功能的判断标准：新手能不能 5 分钟内理解并上手？
2. **能出声就能玩** — 降低从打开到听到第一声音的摩擦
3. **渐进式复杂度** — 基础操作极简，高级功能按需发现
4. **即时反馈** — 操作立刻听到结果，所见即所得

## 🔲 下一阶段：产品打磨（P3）

- [x] **SVG 波形显示** — Clip 内显示波形图形而非纯色块

### P5 — 下一阶段（功能优先）

> 目标：从“能做 demo”升级到“能稳定产出作品”，继续以功能价值优先推进。

#### P5-A（编曲效率）
- [x] **量化/对齐工具（Quantize）** — 对选中 Clip 一键量化到 1/4、1/8、1/16 网格
- [x] **批量编辑（多选 Clip）** — 统一修改音量/淡入淡出/移调/静音
- [x] **区域循环导出（Loop Export）** — 支持导出 loop 区间，而不仅是全曲导出

#### P5-B（音乐性增强）
- [x] **基础鼓机轨（Step Sequencer）** — 提供 16-step 鼓机（Kick/Snare/Hihat）快速搭节奏
- [x] **音阶锁定（Scale Lock）** — 约束音高到指定调式，降低跑调概率

#### P5-C（稳定性与工程化）
- [x] **自动保存版本点（Checkpoint）** — 关键操作自动留最近 N 个版本点，支持快速回退
- [x] **大项目性能模式（Performance Mode）** — 轨道/Clip 较多时自动降级可视渲染，优先播放稳定
- [x] **测试补强（P5 新功能）** — 为 Quantize、Step Sequencer、Scale Lock、Loop Export 补齐 e2e 回归

- [x] **智能补全（Continue）MVP** — 基于当前调式与最近片段补全下一小节；一次提供 3 个候选（保守/平衡/大胆）
- [x] **智能补全（Continue）交互增强** — 增加“再来一版”“锁定节奏/锁定音高”选项，降低试错成本

#### P7-B（一键变好听）
- [x] **Magic Polish（新手混音宏）** — 一键完成基础平衡（音量归一、轻压缩、轻混响、防爆音）
- [x] **Mood Presets（情绪模板）** — 按“开心/治愈/紧张/赛博”等情绪预设自动组合 BPM/调式/音色
- [x] **新手默认安全网** — 默认开启基础限幅与音量保护，避免刺耳和削波

#### P7-C（玩得更有反馈）

### P8 — 下一阶段（功能价值优先）

> 目标：继续提升“从灵感到可分享成品”的完成率，优先补齐最影响成品质量与可复用性的能力。

#### P8-A（成品完成度）
- [x] **母带一键预设（Master Preset Pack）** — 提供至少 4 种可切换母带预设（Clean/Loud/Warm/Bright），应用后实时可听并可一键回退到原始状态；导出音频需写入所选预设结果。
- [x] **导出前响度检查（Loudness Check）** — 导出前显示峰值与整体响度结果，并给出“可直接发布/建议调整”结论；当检测到削波风险时阻止导出并提示可执行修正。
- [x] **片段级包络编辑（Clip Envelope）** — 每个 Clip 支持至少 3 个可拖拽音量包络点（起/中/止），播放时生效且保存后可恢复；支持重置为直线包络。

#### P8-B（编曲效率）
- [x] **轨道冻结（Track Freeze）** — 对含效果器/高负载内容的轨道执行冻结后，播放 CPU 占用可见下降且声音一致；解冻后可继续编辑原始参数。
- [x] **标记点系统（Markers）** — 时间轴支持添加/重命名/删除标记点（如 Intro/Drop），点击标记可跳播到对应小节；项目保存后标记完整保留。

#### P8-C（复用与组织）
- [x] **用户素材收藏夹（Favorite Clips）** — 允许将任意 Clip 存入本地收藏并跨项目复用；收藏项至少包含名称、时长、调式/音高信息，并支持搜索与删除。

#### P8-D（本轮新增需求，功能价值优先）
- [x] **自动混音助手（Auto Mix Assistant）** — 一键执行基础轨道平衡（音量、声像、低频冲突规避），并生成“应用前/应用后”A/B 对比；至少覆盖鼓/贝斯/和声三类轨道，用户可逐项撤销单条建议，导出结果以最终确认版本为准。
- [x] **人声一键清理链（Vocal Clean Chain）** — 对录制人声提供去底噪、齿音抑制、基础压缩与响度补偿的预设链；处理后需在播放中实时生效，并在导出文件中保留；当输入电平过低或削波时给出明确告警与修复建议。
- [x] **参考曲 A/B 对照（Reference Track A/B）** — 允许导入 1 首参考音频并在不进入时间轴编辑的前提下进行 A/B 切听；支持音量自动匹配（±1 dB 内）与快捷键切换，避免“参考曲更响导致误判”。
- [x] **导出清单校验（Pre-export Checklist）** — 导出前自动检查常见失败项（空轨道、静音主总线、峰值削波、未命名项目、循环区与导出区不一致），并给出逐项“通过/未通过”；存在未通过项时禁止一键导出但允许“忽略并继续”。

#### P8-E（新增需求补充，功能价值优先）
- [x] **编排结构助手（Song Arrangement Assistant）** — 基于现有片段一键生成“8/16/32 小节”结构草案（Intro/Verse/Chorus/Drop），自动放置段落标记并复制片段到对应区段；生成后可一键撤销，且保存/重载后段落与片段位置一致。
- [x] **一键分轨导出（Stem Export）** — 支持按轨道批量导出 WAV stems（含主效果链处理后结果），导出包内文件名需包含轨道名与 BPM；当轨道静音或无内容时在导出结果中明确标记“已跳过”。
- [x] **节奏贴合（Audio Beat Align）** — 对导入音频提供“对齐项目 BPM”能力：可选择保持音高或保持时长两种模式；对齐后首拍误差不超过 20ms，并在时间轴显示拉伸状态标识。
- [x] **导出版本对比（Export Version Compare）** — 同一项目支持保留最近 5 次导出记录（时间、格式、时长、响度摘要），可在列表中一键回放对比并重命名版本，帮助新手快速选定“最终版”。
- [x] **发布元信息向导（Release Metadata Wizard）** — 导出前提供标题、作者名、封面、风格标签的必填向导；未填关键字段时不允许进入“分享卡片导出”，填写完成后写入本地项目元信息并可在下次导出复用。

#### P8-F（新增需求补充，功能价值优先）
- [x] **人声对齐与节拍校正（Vocal Timing Align）** — 对录制人声提供“一键贴合网格”与“按小节轻度拉伸”两档校正；校正后可实时 A/B 对比，且任意片段可单独回退；验收标准：在 120 BPM、8 小节样例中，主要发音起点偏差收敛到 ±30ms 内。
- [x] **副歌增强器（Chorus Lift Builder）** — 一键为选定副歌区段生成“能量提升”处理（鼓层加密、和声加厚、自动增益上扬 1-2 dB）；处理范围仅限标记区段且可逐项关闭；验收标准：开启后导出版本在对应区段的平均响度高于主歌区段且无削波。
- [x] **项目健康面板（Project Health Panel）** — 提供实时可见的项目风险列表（CPU 过载、静音但有内容轨道、未命名标记、导出区间异常）；每项风险可点击跳转到对应位置并给出修复动作；验收标准：面板内风险状态与导出前检查结果一致。
- [x] **新手可读混音报告（Mix Report for Beginners）** — 在导出后自动生成一页报告（轨道峰值、响度分布、建议动作）并附“下一步优化”按钮；报告需支持保存到项目历史并可对比最近一次导出报告。

#### P8-G（新增需求补充，功能价值优先）
- [x] **人声一键音高校正（Vocal Pitch Assist）** — 对选中人声 Clip 提供“轻度自然/流行明显”两档音高修正，默认按项目调式与音阶锁定处理；支持实时开关与 Dry/Wet 混合；验收标准：在 16 小节演唱样例中，离调音符占比相对原始版本下降 ≥60%，且导出结果与实时回放一致。
- [x] **副歌堆叠生成器（Chorus Double & Harmony）** — 在选定副歌区段一键生成人声 Double 与三度和声（可选高八度），自动与主唱对齐并独立电平控制；验收标准：生成后至少包含 2 条新轨（Double/Harmony），可单独静音对比且无明显相位抵消（主唱+Double 峰值下降不超过 2 dB）。
- [x] **导出目标预设（Export Target Presets）** — 导出时提供“短视频平台/播客/音乐平台/通用”目标预设，自动设置采样率、码率、响度目标与峰值上限，并展示与当前设置差异；验收标准：切换预设后参数即时生效，导出文件元数据与目标参数一致，且可一键恢复自定义设置。
- [x] **项目恢复中心（Recovery Center）** — 提供崩溃恢复与误操作恢复入口，展示最近自动保存快照与导出前快照，可预听并恢复为新副本；验收标准：异常关闭后重开应用可看到最近 5 个可恢复点，恢复成功率 100%，且不覆盖当前项目。
- [x] **母带参考匹配（Reference Match Master）** — 允许选择“本项目历史导出”或“参考曲”作为匹配目标，自动建议 EQ 倾向与动态范围调整并可分项应用；验收标准：应用后频谱差异（低/中/高三段平均偏差）较应用前下降，且任一建议可单独撤销。

#### P8-H（新增需求补充，功能价值优先）
- [x] **一键人声成品链（Vocal Finalizer）** — 对选中人声轨提供“清晰/暖声/贴耳”三档成品处理链（EQ、压缩、去齿音、空间），支持实时 A/B 与强度滑杆（0-100%）；验收标准：开启后导出结果与实时监听一致，且在官方人声样例中主观清晰度评分较原始版本提升（≥80% 评测样本给出“更清晰”结论）。
- [x] **段落能量自动化（Section Energy Automation）** — 基于 Marker 段落（Verse/Chorus/Drop）一键生成主总线与关键轨道的能量曲线（音量、滤波开度、空间量），并允许按段落单独启用/禁用；验收标准：自动化仅作用于选定段落范围，回放与导出保持一致，且可一键恢复到“无自动化”初始状态。
