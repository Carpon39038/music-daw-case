# 🎯 music-daw-case 产品文档

> **定位**：面向零基础用户的浏览器端 DAW，帮助新手 5 分钟内做出自己的小 demo。

---

## 核心原则

1. **新手优先** — 每个功能的判断标准：新手能不能 5 分钟内理解并上手？
2. **能出声就能玩** — 降低从打开到听到第一声音的摩擦
3. **渐进式复杂度** — 基础操作极简，高级功能按需发现
4. **即时反馈** — 操作立刻听到结果，所见即所得

## 🔲 下一阶段：产品打磨（P3）

> P0-P2 功能已全部就绪，代码约 4400 行、78 个 e2e 测试。接下来围绕**"新手能真正做出完整 demo 并分享"**这个终极目标打磨体验。

### P3-A — 分享 & 完整性（达成"做出 demo 并分享"闭环）
- [x] **项目保存/加载** — localStorage 持久化，关闭浏览器不丢进度
- [x] **分享链接** — 将项目编码到 URL hash 或生成短链接，发给朋友能直接打开听到
- [x] **MP3 导出** — 除 WAV 外支持 MP3 格式，方便社交分享

### P3-B — 新手引导深化
- [x] **引导完成后的"下一步提示"** — 新手完成引导后，引导他加载第一个 demo 模板试试
- [x] **空白项目时的引导气泡** — 新建项目后如果 10 秒没操作，弹出轻量提示
- [x] **快捷键面板** — 界面上有 `?` 按钮，点击显示所有快捷键一览

### P3-C — 音频体验提升
- [x] **更丰富的 demo 模板** — 至少 4-5 个不同风格的预设（电子/嘻哈/氛围/古典/Lo-Fi）
- [x] **音色预览** — 在音色选择器悬停时播放该音色的示范音
- [x] **录音后自动创建 Clip** — 录音结束后自动将录音片段放到当前轨道的时间轴上（目前录音功能缺少与时间轴的整合）

### P3-D — 视觉 & 交互优化
- [x] **SVG 波形显示** — Clip 内显示波形图形而非纯色块
- [x] **轨道颜色系统** — 每条轨道可设置颜色，Clip 背景用 track.color 30% 半透明
- [x] **拖拽反馈优化** — 拖拽 clip 时显示吸附位置预览
- [x] **响应式布局** — 支持常见笔记本屏幕尺寸（1366px+）

### P4 — 功能优先迭代路线（下一阶段）

> 延续“功能开发优先、交互优化次之”的策略，优先补齐影响成品完成度与可用性的能力。

#### P4-A（高优先级）
- [x] **拖拽反馈可视化** — 拖拽 Clip 时显示 ghost clip、吸附线、目标轨道高亮，提升放置可预期性
- [x] **桌面响应式适配（1366px+）** — 在 1366/1440 常见分辨率下保证 Transport/Timeline/Inspector 布局稳定可用

#### P4-B（功能闭环增强）
- [x] **音频文件导入（最小可行版）** — 支持将本地音频文件拖入时间轴并自动创建 Clip
- [x] **项目元信息管理（轻量）** — 项目名、最后保存时间、另存为副本

#### P4-C（质量与发布准备）
- [x] **大项目性能稳定性优化** — 多轨/多 Clip 场景下播放与拖拽流畅度优化
- [x] **测试补强** — 为拖拽反馈、响应式补齐 e2e 覆盖，降低回归风险
- [x] **测试补强 (音频导入)** — 为音频导入补齐 e2e 覆盖

### P5 — 下一阶段（功能优先）

> 目标：从“能做 demo”升级到“能稳定产出作品”，继续以功能价值优先推进。

#### P5-A（编曲效率）
- [x] **量化/对齐工具（Quantize）** — 对选中 Clip 一键量化到 1/4、1/8、1/16 网格
- [x] **批量编辑（多选 Clip）** — 统一修改音量/淡入淡出/移调/静音
- [x] **区域循环导出（Loop Export）** — 支持导出 loop 区间，而不仅是全曲导出

#### P5-B（音乐性增强）
- [x] **基础鼓机轨（Step Sequencer）** — 提供 16-step 鼓机（Kick/Snare/Hihat）快速搭节奏
- [x] **音阶锁定（Scale Lock）** — 约束音高到指定调式，降低跑调概率

#### P5-C（稳定性与工程化）
- [x] **自动保存版本点（Checkpoint）** — 关键操作自动留最近 N 个版本点，支持快速回退
- [x] **大项目性能模式（Performance Mode）** — 轨道/Clip 较多时自动降级可视渲染，优先播放稳定
- [x] **测试补强（P5 新功能）** — 为 Quantize、Step Sequencer、Scale Lock、Loop Export 补齐 e2e 回归

### P6 — 下一阶段（功能优先）

> 目标：从"能稳定产出作品"升级到"更有音乐性、更易分享"，继续以功能价值优先推进。

#### P6-A（音乐性提升）
- [x] **和弦预设** — 一键插入常用和弦进行（如 I-V-vi-IV），替代逐音调整
- [x] **旋律生成器** — 基于音阶锁定随机生成旋律片段，给新手灵感
- [x] **音频 Normalize** — 一键均衡所有 Clip 音量，解决新手最常遇到的各轨音量不均问题
- [x] **节拍变化（速度曲线）** — 同一首歌内变速（渐快/渐慢），让作品更有表现力

#### P6-B（分享与作品化）
- [x] **项目模板保存** — 把自己做的 beat 存为模板，下次复用
- [x] **作品集/画廊** — 本地展示所有保存的项目，像一个小作品集
- [x] **一键发布到社交媒体** — 导出 + 生成带封面的小视频/音频卡片

#### P6-C（效率与体验）
- [x] **右键菜单** — Clip/轨道右键弹出操作菜单（复制/删除/移调等），比快捷键更直观
- [x] **跨轨道复制** — 把一个 Clip 复制到另一个轨道的不同位置
- [ ] **和弦检测/建议** — 根据已有音符自动推荐搭配的音

### P7 — For Fun 增强（新手快乐优先）

> 目标：让音乐小白“更快出结果、持续有成就感”，突出好玩而非专业深度。

#### P7-A（先把“会玩”做出来）
- [ ] **风格一键生成（Style Starter）** — 选择 Lo-Fi/EDM/HipHop 等风格，一键生成可直接播放的 8 小节草稿（鼓+和弦+bass）
- [ ] **智能补全（Continue）MVP** — 基于当前调式与最近片段补全下一小节；一次提供 3 个候选（保守/平衡/大胆）
- [ ] **智能补全（Continue）交互增强** — 增加“再来一版”“锁定节奏/锁定音高”选项，降低试错成本

#### P7-B（一键变好听）
- [ ] **Magic Polish（新手混音宏）** — 一键完成基础平衡（音量归一、轻压缩、轻混响、防爆音）
- [ ] **Mood Presets（情绪模板）** — 按“开心/治愈/紧张/赛博”等情绪预设自动组合 BPM/调式/音色
- [ ] **新手默认安全网** — 默认开启基础限幅与音量保护，避免刺耳和削波

#### P7-C（玩得更有反馈）
- [ ] **30 秒出歌挑战模式** — 三步流程（选风格→改一点→导出）快速闭环
- [ ] **成就反馈系统（轻量）** — 首次导出、首次用和弦、首次完成 16 小节给即时反馈/徽章
- [ ] **作品卡片导出（本地）** — 导出带封面与标题的分享卡片（先本地文件，不做平台直发）
