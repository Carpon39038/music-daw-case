# 🎯 产品目标

**music-daw-case** 是一个面向新手的浏览器端 DAW（数字音频工作站），帮助零基础用户入门音乐制作，做出自己的小 demo。

## 核心原则

1. **新手优先** — 每个功能的判断标准：新手能不能 5 分钟内理解并上手？
2. **能出声就能玩** — 降低从打开到听到第一声音的摩擦
3. **渐进式复杂度** — 基础操作极简，高级功能按需发现
4. **即时反馈** — 操作立刻听到结果，所见即所得

## 设计决策：现有功能取舍

**原则：不删底层能力，加新手友好的翻译层 + 隐藏高级功能**

### 保留且优化呈现
| 现有 | 改进方向 |
|------|---------|
| Clip 音高（Hz 输入） | 改为音符选择器（C3、D4、A5…），底层仍用频率 |
| Clip 音量（gain dB） | 改为百分比滑块（0-100%），直观 |
| Clip 淡入淡出（ms） | 保留但默认折叠，不暴露给新手 |
| 波形类型（sine/square…） | 改为音色名称（柔和/锐利/温暖/电子），映射到底层波形 |
| 轨道音量/静音/Solo | 保留，已经够直观 |
| Undo/Redo、复制粘贴 | 保留，很好的基础功能 |
| 循环播放、节拍器 | 保留，新手需要 |
| Clip 拖拽、拆分 | 保留，已经够直观 |

### 隐藏到"高级"折叠区
| 功能 | 原因 |
|------|------|
| 3 段 EQ | 新手不需要手动调频段 |
| MIDI 导入/导出 | 新手不知道 MIDI 是什么 |
| Transpose（半音移调） | 可保留但默认不展示 |
| Clip scheduled frequency 显示 | 纯技术信息，隐藏 |

### 新增功能（按优先级）

#### P0 — 上手第一印象
- [x] **预设 demo 模板** — 内置 2-3 个小项目（鼓点节奏、简单旋律），新手打开直接播放
- [ ] **新手引导** — 首次打开的交互式引导（添加轨道→选音色→播放）
- [x] **键盘快捷键** — Space 播放/暂停，Ctrl+Z 撤销，界面内提示
- [x] **点击 clip 即刻预览** — 不用等播放就能听到声音

#### P1 — 核心体验
- [ ] **音符选择器替代频率输入** — 下拉选 C3/Bb4 等，不手动输 Hz
- [ ] **音色名称替代波形名** — "柔和/锐利/温暖/电子"替代 sine/square
- [ ] **音量百分比滑块** — 替代 gain dB 数字输入
- [ ] **导出音频** — WAV/MP3，新手做完 demo 能分享
- [ ] **播放高亮** — 播放时高亮当前 clip 和 beat

#### P2 — 进阶
- [ ] **简单效果器预设** — 混响/延迟一键开关，不需要手动调参数
- [ ] **录音功能** — 用麦克风录制
- [ ] **更多预设音色** — 扩充内置合成器音色

## 不做

- ❌ 专业级功能（自动化曲线、高级 MIDI 编辑、多声道输出）
- ❌ 插件系统（VST/AU）
- ❌ 移动端适配（专注桌面浏览器）
- ❌ 协作/云存储
